Compiler Design Lab Works

Lab assignments for a compiler design course. Each lab explores a core concept in compiler construction, lexical analysis, parsing, code generation, etc.

## Labs

### Lab 1: Lexical Analysis (Tokenizer)
Implements a tokenizer for lexical analysis. Identifies and classifies tokens into keywords, delimiters, and operators. Reads source code and produces a token stream.

**Run:** `python lab1.py`

### Lab 2: Regular Expression Matching
Implements regular expression pattern matching supporting operators like `*`, `+`, and `?`. Tests pattern matching against input text.

**Run:** `python lab2.py`

### Lab 3: Left Recursion Removal
Implements algorithms to eliminate left recursion from context-free grammars. Transforms recursive grammar rules into equivalent non-recursive forms.

**Run:** `python lab3.py`
